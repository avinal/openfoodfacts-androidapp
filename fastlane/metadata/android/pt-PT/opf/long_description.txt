O Open Food Facts é uma base de dados de produtos alimentares com ingredientes, alergénios, tabela nutricional e todos os pedaços de informação que se encontram nos rótulos dos produtos.

***Contribuir para a transparência dos alimentos para animais ***

- Open Product Facts é um projeto não lucrativo desenvolvido por milhares de voluntários em todo o mundo.
- Pode usá-lo para fazer melhores escolhas alimentares para animais e como são dados abertos, qualquer um pode reutilizá-lo para qualquer finalidade.
- Open Pet Food Facts é um projeto sem fins lucrativos desenvolvido por milhares de voluntários em todo o mundo. Temos muitos projetos interessantes para os quais você pode contribuir de diversas maneiras. 
- Pode começar a contribuir hoje, adicionando um produto do seu armário de alimentos para animais.

*** Listagem original do dia das mentiras ***

Ontem de manhã, a equipa do Open Food Facts acordou para uma revelação impressionante. 

Colecionar alimentos, cosméticos e alimentos para animais de estimação no mundo estava tão fora de moda. Com o imenso progresso feito no ano passado, certamente devemos estar próximos de completar a nossa missão de colecionar todos os produtos alimentares e cosméticos do mundo, não é ?

Por isso, criámos não 1, mas outras 77000 novas base de dados separadas! 

O Open Products Facts será um projeto extremamente ambicioso. Eis alguns deles: 

Open Crayons Facts - A maior base de dados aberta de lápis de cor

Open Miniature Food Facts - A maior base de dados aberta de produtos alimentares em miniatura

Aberto??? Factos - a maior base de dados de quê???

Amanhã vamos selecionar as 3 base de dados mais bem sucedidas e vamos começar a colecionar todos os produtos correspondentes no mundo.

Serão bases de dados de latas de lixo, cartões SIM ou telemóveis? 

Por isso, se gostas de t-shirts, livros de culinária, papel de origami ou globos de neve, comeca a digitalizar usando a aplicação Open Products Facts Android no Google Play
